# Incident Response Module

This directory contains the tools and templates for handling production incidents effectively.

## Core Documents
*   **[checklist.md](./checklist.md):** Interactive checklist for the Incident Commander (IC).
*   **[severity-levels.md](./severity-levels.md):** Definitions of SEV-1 to SEV-4 levels.
*   **[postmortem-template.md](./postmortem-template.md):** Standard template for Retrospectives/Postmortems.
*   **[sla-tiers.md](./sla-tiers.md):** Service Tiers (A/B/C) and Incident Response SLAs.

## Automation
*   **`config/metadata.json`:** Configuration file for the postmortem generator (Service name, Emails, Tags).
*   **`postmortems/`:** Directory where generated postmortem files are stored.
